## Get the List of containers and associated Softwares

This is created with `pyhton`.Prerequisite please install the requirements
```
pip install -r requirements.txt
```

#### Please update the config.json
- `api_gateway_url` can be located at https://www.qualys.com/platform-identification/
- `api_access_token` can be located on Qualys Container Security Platform . `Configurations > Access Token`
- `container_state` accepts `['RUNNING', 'ALL']`
